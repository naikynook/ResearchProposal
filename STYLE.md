# Website Style Direction

## Positioning
Thought repository for Smart Room. Light enough to feel immersive and enterable, but **not commercial product UI**. The stance is **sci-fi / alien world-building**: off-world chamber, bioluminescence, organic geometry—with a quiet hacktivist extraction signal underneath.

## Dual Stance
1. **Alien surface (primary):** pale green-mist grounds, chartreuse / teal biolight, Michroma titles, irregular spores, orbital rings.
2. **Extraction layer (secondary):** crimson on section indices only—data capture as foreign signal inside the beautiful world.

## Visual Character
- **Mood:** otherworldly biome / monumental sci-fi space—not startup landing page.
- **Not:** SaaS blues, soft marketing orbs, purple gradients, cream editorial, dark cyber HUD.
- **Yes:** mineral greens, acid alien accents, crystalline hero frame, pentagon brand mark, clipped keyword chips, noise grain.

## Color
| Role | Direction |
|------|-----------|
| Ground | Pale alien mist (`#e6ece7`) |
| Text | Deep forest ink (`#0e1a14`) |
| World | Bioluminescent teal / acid green |
| Extract | Crimson signal (`#c41e4a`) sparingly |

## Typography
- **Display:** Michroma — rigid sci-fi monumentality
- **Body:** Chakra Petch — angular, technical, non-corporate
- **Meta:** Share Tech Mono — system / hacktivist layer

## Motion
Spore drift, slow orbital rings, mesh drift, hero rise-in, active-section spy.
